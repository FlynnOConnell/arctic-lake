### Registering with Scipy vs diSPIMFusion

Both methods start the same:

Parse XML, compose full transform chain per setup (registration @ manual @ calibration)
Compute T_rel = inv(T_full_spima) @ T_full_spimb — the exact BigStitcher mapping
For scipy (works in raw voxel space):
3. Swap X↔Z axes with permutation P (BDV is XYZ, numpy is ZYX)
4. Pass inv(T_rel_zyx) to affine_transform

For diSPIMFusion (works in isotropic space):
3. Convert to iso: T_rel_iso = S @ T_rel @ inv(S) where S = diag(1,1,pz/pxy,1)
4. Build centered rotation: R_centered = T(center) @ R90Y @ T(-center)
5. Invert: TMX = R_centered @ inv(T_rel_iso)
