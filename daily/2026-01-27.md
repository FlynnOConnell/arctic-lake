Metadata (req):

- dimensions:	768x1848x38,768x1848x38	Image size (W×H×Z). Parsed to get zplanes=38
- z_step:	6.220	Physical spacing between Z-planes (micrometers)
- exposure_time:	0.7	Exposure in ms. Used to calculate fps = 1000/0.7 ≈ 1428 and vps = fps/zplanes
- detection_objective:	Zeiss 20x/1.00	Magnification extracted via regex → objective_mag=20.0

Derived:

- zplanes = 38  # from dimensions
- fps = 1000 / 0.7  # ~1428 frames/sec
- vps = fps / zplanes  # ~37.6 volumes/sec
- pixel_resolution_um = 6.5 (hardcoded) / objective_mag (20.0)  # = 0.325 µm/pixel
