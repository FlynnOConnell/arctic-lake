## ImageJ "hyperstack"

ImageJ metadata tags tell readers how to interpret the flat pages:

frames = number of T timepoints
slices = number of Z planes
channels = 1 (grayscale)
images = total pages (T × Z × C)
hyperstack = True
Your code does this:

reshape to TZCYX for imagej (add C=1)
```python
chunk_5d = chunk_data[:, :, np.newaxis, :, :]  # (T, Z, C, Y, X)
```

write each frame (T) with all its Z slices

```python

for t in range(chunk_5d.shape[0]):
  frame_data = chunk_5d[t]  # (Z, C, Y, X)
  writer.write(frame_data, ...)  # writes Z×C pages

```
