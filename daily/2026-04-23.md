## Notes

Filters you cannot disable:

- snr threshold:  https://github.com/MouseLand/suite2p/blob/e6ed85fc81f4d0660e8f3c783a0eb58a4b8c1b4e/suite2p/pipeline_s2p.py#L190
- classification: https://github.com/MouseLand/suite2p/blob/e6ed85fc81f4d0660e8f3c783a0eb58a4b8c1b4e/suite2p/pipeline_s2p.py#L238

Preclassify drop (optional, off by default) — removes ROIs by classifier prob before extraction:
https://github.com/MouseLand/suite2p/blob/e6ed85fc81f4d0660e8f3c783a0eb58a4b8c1b4e/suite2p/detection/detect.py#L248

npix_norm bounds — size filter on normalized pixel count:
https://github.com/MouseLand/suite2p/blob/e6ed85fc81f4d0660e8f3c783a0eb58a4b8c1b4e/suite2p/detection/stats.py#L225

max_overlap drop — removes ROIs sharing too many pixels with higher-variance ROIs:
https://github.com/MouseLand/suite2p/blob/e6ed85fc81f4d0660e8f3c783a0eb58a4b8c1b4e/suite2p/detection/stats.py#L244

soma_crop — crops dendrites before computing compact, aspect_ratio, npix_soma (the features that feed the classifier):
https://github.com/MouseLand/suite2p/blob/e6ed85fc81f4d0660e8f3c783a0eb58a4b8c1b4e/suite2p/detection/stats.py#L93

Hardcoded 0.5 classifier threshold — the actual cell/not-cell decision boundary; your classification link covers the call site, but this is where the cutoff lives:
https://github.com/MouseLand/suite2p/blob/e6ed85fc81f4d0660e8f3c783a0eb58a4b8c1b4e/suite2p/classification/classifier.py#L78

Classifier file selection — priority order (classifier_path → builtin → user-trained); affects what the final classification actually does:
https://github.com/MouseLand/suite2p/blob/e6ed85fc81f4d0660e8f3c783a0eb58a4b8c1b4e/suite2p/pipeline_s2p.py#L93

Red/chan2 labeling — separate redcell.npy, doesn't affect iscell but is a component filter:
- intensity-based: .../suite2p/detection/chan2detect.py#L156
- cellpose IoU: .../suite2p/detection/chan2detect.py#L210

If you want to rank by impact for a typical run: classification (always on) > max_overlap (always on, default 0.75) > soma_crop (shapes classifier inputs) > npix_norm >
 snr_threshold (off by default) > preclassify (off by default).
