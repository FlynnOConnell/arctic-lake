# Project Guidelines

## For each Project

- Have a `.PPTX` for short-form summary of steps and results (OneDrive)
- An accompanying Project Folder with Project Notes for long-form contents (Storage Server)

## Project Notes

- MUST detail ALL steps
- MUST be clear for another team member to follow and replicate
- MUST provide path to the referenced data on the Servers

### Permitted Formats

- Markdown
- MS Office (`.docx`, `.xlsx`)
- PDF
- HTML
- TXT
- `.ipynb` - save a static version (render as PDF or HTML)

### Not Permitted

- Personal Dropbox
- Google Drive
- Links to external apps or similar

## Project Folder

Add all relevant:

- Project Notes
- (Internal) How-To-Guides
- Knowledgebase
- Protocols

Port over notes (e.g., literature review summaries, Suite3D, EXTRACT, ActSort, and other such excursions)
